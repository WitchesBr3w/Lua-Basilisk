To make this script work, you already need a few pre existing gun sounds and animations within your gun model  aswell as a remote event named "Fire"
This script also assumes that you already have some sort of currency system within your game (as evident by the Cash and Killstreak values it tries to obtain)
The "GuiModule" is just a module script which you put under ReplicatedStorage, all it really does it move the position of a textlabel up or alternatively does a simple typerwiter effect on a textlabels content
Finally, put the cilent script file as a local script and the server script file as well a regular script both under the gun tool
